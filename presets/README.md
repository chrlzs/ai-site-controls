# `robots.txt` Presets

Copy one of these files to the published web root of your site as `robots.txt`.

- `conservative/` blocks AI-specific bots where possible
- `balanced/` allows search and user-directed retrieval, but blocks training
- `open/` allows broad AI access, including training bots

The root `public/robots.txt` matches the balanced preset.

Use `../crawler-matrix.md` for sourced notes on why the presets differ.
